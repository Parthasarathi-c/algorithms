Matrix Chain Multiplication – Bottom‑Up DP

Computes the minimum scalar multiplications to multiply n matrices using the classic MATRIX‑CHAIN‑ORDER algorithm.

MATRIX-CHAIN-ORDER(p, n):            // p[0..n] dims, n matrices
    for i ← 1 to n:
        m[i][i] ← 0                  // cost of single matrix

    for len ← 2 to n:                // sub‑chain length
        for i ← 1 to n - len + 1:
            j ← i + len - 1
            m[i][j] ← ∞
            for k ← i to j - 1:      // split point
                cost ← m[i][k] + m[k+1][j] + p[i-1]*p[k]*p[j]
                if cost < m[i][j]:
                    m[i][j] ← cost   // best cost for (i..j)
                    s[i][j] ← k      // record split

    return m[1][n], s                // optimal cost & split table

Time Θ(n³), space Θ(n²). Use table s to print the optimal parenthesisation.
