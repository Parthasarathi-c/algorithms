Rod Cutting O(2ⁿ)

This pseudocode explores every possible first cut to maximise revenue, leading to exponential time. (The DP bottom‑up variant is Θ(n²).)

rod_cut(p, n):
    if n == 0:
        return 0                      // base: empty rod

    q = -∞                            // best revenue so far
    for i = 1 to n:                   // try first cut of length i
        // revenue = price of length‑i piece + best of remainder
        q = max(q, p[i-1] + rod_cut(p, n-i))

    return q

Time ≈ Θ(2n) – each call branches on up to n first‑cut choices. Optimise by memoising results (top‑down DP) or filling a table bottom‑up (Θ(n²)).
