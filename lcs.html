Longest Common Subarray

This pseudocode outlines the process to find the longest common subarray between two strings using dynamic programming.

procedure LCS-Length (X, Y) 
begin 
    m = X.length; n = Y.length 
    for i = 1 to m c[i, 0] = 0; 
    for j = 0 to n c[0,  j] = 0; 
    for i = 1 to m 
        for j = 1 to n 
            if (x[i] == y[j]) 
                c[i, j] = c[i-1, j-1] + 1; b[i, j] = “↖”; 
            else if (c[i-1, j] ≥ c[i, j-1]) 
                c[i, j] = c[i-1, j]; b[i, j] = “↑”; 
            else 
                c[i, j] = c[i, j-1]; b[i, j] = “←”; 
    return c and b 
end
