Quickselect – Selection in Expected O(n) Time

This pseudocode outlines the randomized Quickselect algorithm used to find the k‑th smallest element in expected linear time.

selection(A, low, high, k):
    if low == high: return A[low]               // only one item

    q = partition(A, low, high)                // Hoare/Lomuto split
    x = q - low + 1                            // rank of pivot in this slice

    if q == k:               return A[q]       // pivot is the answer
    else if q > k:           return selection(A, low, q‑1, k)      // left side
    else                     return selection(A, q+1, high, k‑x)   // right side

Here, partition reorders the slice so that all elements < pivot come before index q, and those > pivot come after. Only the relevant side is recursed into, yielding an average recurrence of T(n) = T(n/2) + Θ(n) ⇒ expected Θ(n).
