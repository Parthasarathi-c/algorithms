Naïve String Matching – CS23404

A brute‑force pattern search that checks every alignment of pattern P in text T.

NAIVE-STRING-MATCHER(T, P, n, m):
    for s ← 0 to n − m do                 // all shifts
        if P[1..m] == T[s+1 .. s+m] then
            print "pattern occurs with shift", s

Worst‑case time Θ(n·m); space Θ(1). Good baseline before KMP or Boyer‑Moore.
