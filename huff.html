Huffman Coding Algorithm

This pseudocode outlines the Huffman Coding algorithm used for data compression, which is an optimal way of encoding characters based on their frequencies.

procedure HUFFMAN(C, f):
    Q ← empty min-heap                // priority queue of nodes

    for each character c in C do
        create node z
        z.char ← c
        z.freq ← f[c]
        z.left ← NIL
        z.right ← NIL
        INSERT(Q, z)                  // insert into priority queue

    while SIZE(Q) > 1 do
        x ← EXTRACT_MIN(Q)           // lowest frequency node
        y ← EXTRACT_MIN(Q)           // next lowest frequency
        create node z
        z.char ← NIL
        z.freq ← x.freq + y.freq
        z.left ← x
        z.right ← y
        INSERT(Q, z)                 // push merged node back

    return EXTRACT_MIN(Q)            // root of final Huffman tree

procedure EXTRACT_MIN(Q):
    if Q.size == 0:
        return "Heap is empty"  // Return a message if the heap is empty
    
    min_element = Q[0]  // The root (smallest element) is the minimum element
    Q[0] = Q[Q.size - 1]  // Replace the root with the last element
    Q.size = Q.size - 1  // Decrease the size of the heap

    // Restore the heap property by heapifying down from the root
    heapify_down(Q, 0)  // Start from the root (index 0)
    
    return min_element

The Huffman Coding algorithm builds an optimal binary tree where the characters with higher frequencies are closer to the root, resulting in shorter codes for frequent characters.
