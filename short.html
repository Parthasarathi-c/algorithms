Dijkstra’s Algorithm – Single Source Shortest Path

Finds the shortest path from a source node to all other nodes in a graph with non-negative edge weights.

dijkstra(G, w, s):
    for each vertex v in G:
        dist[v] ← ∞
        prev[v] ← NULL
    dist[s] ← 0
    Q ← all vertices in G

    while Q ≠ ∅:
        u ← vertex in Q with smallest dist[u]
        remove u from Q

        for each neighbor v of u:
            alt ← dist[u] + w(u, v)
            if alt < dist[v]:
                dist[v] ← alt
                prev[v] ← u

    return dist, prev

The algorithm builds the shortest paths from a source node s using a priority queue.
Time complexity: O((V + E) log V) with a binary heap.
